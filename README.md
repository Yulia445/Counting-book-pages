# Counting-book-pages

## Опис

Цей код працює з текстовим файлом book.txt і ділить його на сторінки. Спочатку ми читаємо весь файл у змінну data. Одна сторінка має фіксовану кількість символів — 120 × 80. Функція countPages(symbolsInBook, symbolsOnPage) рахує, скільки всього сторінок буде в книжці. Вона перевіряє, чи правильно передані числа, і повертає округлену вгору кількість сторінок. Функція getPage(book, pageNumber) визначає, які символи відповідають потрібній сторінці. Щоб не обривати слова, вона шукає найближчий пробіл або перенос рядка перед кінцем сторінки. Якщо знаходить — обрізає текст до цього місця. Функція pagesText(data, pageNumber) перевіряє, чи існує така сторінка. Якщо номер неправильний — повертає повідомлення, скільки сторінок є. Якщо все ок — повертає текст конкретної сторінки.
В кінці код виводить у консоль текст 300-ї сторінки або повідомлення, що її немає.
